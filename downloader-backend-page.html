NKP Downloader

Pick a target version, paste a download.nutanix.com signed link, enqueue. Downloads run one at a time straight onto Harbor in the version folder. The URL clears after each enqueue so you can stack the next one.

Downloads run one at a time (sequential queue). Only https://download.nutanix.com/ links are accepted. Each file lands in its version folder under /data/nkp-sources/ — the bastion sees them over NFS at /mnt/nkp-sources/. After a file finishes, use its push button for the exact nkp push bundle command (image bundles only).
